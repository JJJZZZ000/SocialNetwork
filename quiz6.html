Lecture #6 Quiz

Modify this <div> so that its width is most of the screen (80% to 90%). And, make its width, font-size, margin, padding, and border grow when the browser's viewport width gets bigger, and shrink when the viewport width gets smaller so that this <div> always fits in the page.

    • The <div> must span most of the page width
    • The <div> width must scale (grow/shrink) when the viewport width changes
    • The font-size of the letters in the <div> must scale when the viewport width changes
    • The margin, padding, and border for the <div> must scale when the viewport width changes
